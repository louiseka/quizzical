## What is Quizzical?
**Quizzical** is a 5-question general knowledge quiz. Questions are dynamically generated using the **Open Trivia Database API**.  

This project focuses on **React component design**, state management, API integration, and testing with Vitest.

## Features
- Reusable components for each question and answer  
- Reusable homepage component  
- State management for tracking quiz progress  
- `useEffect` to manage side effects based on dependencies  
- Basic tests implemented using **Vitest**

## Technologies Used
- **React** – Component-based UI  
- **JavaScript** – Logic and interactivity  
- **CSS** – Styling  
- **Vitest** – Unit testing  

## Future Improvements
- Allow users to select the **type of quiz** (category/topic)  
- Let users choose **number of questions** for their quiz  

## Installation (Local Development)
To run the project locally:

```bash
# Clone the repository
git clone https://github.com/yourusername/quizzical.git

# Navigate into the project directory
cd quizzical

# Install dependencies
npm install

# Start the development server
npm run dev
```
